# HistorySearch
HistorySearch is a java program that allows users to search for predefined historical events and read detailed descriptions.
## How To Run
1. Download or clone this repository.
2. Open the project in a Java IDE.
3. Run 'Main.java'
4. Enter a keyword such as '300' when prompted.
## Example Event
 - 300 -> Battle of Thermopylae
